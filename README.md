# 大型测试仓库

这是一个用于AICR-Agent性能测试的大型测试仓库。

## 结构

- `src/`: 源代码
  - `api/`: API相关代码
  - `core/`: 核心功能
  - `utils/`: 工具函数
  - `models/`: 数据模型
  - `tests/`: 测试用例
- `docs/`: 文档
- `config/`: 配置文件
- `scripts/`: 工具脚本
- `assets/`: 二进制资源
- `third_party/`: 第三方库
- `examples/`: 示例代码
- `web/`: Web相关代码
- `mobile/`: 移动端相关代码

## 用途

此仓库仅用于测试目的。
